## Big Data

### Only information part
The summary will be as long as much users there are. So if you only want to see the information header, you can use the `-i` argument.

`telestats.py log/bigdata -i`
```
Rows:            1310817
Loading:         100.0%

Start:           2018-05-13 19:50:42
End:             2018-05-14 21:15:25
Duration:        1 day, 1:24:43
Users:           26245
Online:          780
```

### Duration information
Prints quick information about the start- and endtime and the duration of the capture. This is a lot faster than `-i` in case of a big capture file because only the first and last 1000 lines are read.

`telestats.py log/bigdata --durationinfo`
```
Rows:            1304242
Loading:         100.0%

Start:           2018-05-13 19:50:42
End:             2018-05-14 21:06:37
Duration:        1 day, 1:15:55
```

### Get username
In the summary are all users listed by the printname. If you wanna get the username, you can resolve the name for the username with the `--username` paramenter.

`telestats.py log/bigdata --username ThisIsMyPrintname`
```
Rows:            1330644
Loading:         34.1%

Name:           ThisIsMyPrintname
Username:       thisismyusername
```
